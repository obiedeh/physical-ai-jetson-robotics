# Robotics Evidence Reports

This folder is for robotics validation artifacts that show the ROS 2, simulation, robot description, and sim-to-real paths working.

Expected artifacts include:

- ROS 2 launch screenshots
- RViz screenshots
- TF tree
- joint states
- URDF/USD validation notes
- sim-to-real notes

Use this folder for evidence, not plans. If a robot path is still scaffolded, label it as scaffolded and describe the next validation step.
