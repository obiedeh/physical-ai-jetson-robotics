# LeRobot Project Track

This directory contains repo-local configuration and notes for LeRobot, ALOHA-compatible datasets, and Synria arm learning workflows.

Primary guide:

```text
docs/LEROBOT_ALOHA.md
```

Generated datasets and checkpoints are intentionally not stored here. Use:

```text
data/lerobot/
```

or Hugging Face repositories for datasets/models that should be shared.
